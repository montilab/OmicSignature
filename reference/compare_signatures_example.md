# Example signatures for comparing OmicSignature lists

A named list of four \`OmicSignature\` objects prepared for examples and
vignettes that demonstrate \`compare_omic_signatures()\` and
\`signature_similarity_heatmap()\`. The objects come from local
signature RDS files used in the comparison workflow and have their
\`difexp\` tables reduced to the union of features present in the stored
signatures to keep the package data footprint small.

## Usage

``` r
data(compare_signatures_example)
```

## Format

A named list with four \`OmicSignature\` objects:

- e7386_hsc3:

  E7386 versus DMSO signature for HSC3 cells.

- e7386_cal27:

  E7386 versus DMSO signature for CAL27 cells.

- icg001_hsc3:

  ICG001 signature for HSC3 cells.

- icg001_cal27:

  ICG001 signature for CAL27 cells.

## Source

Generated by \`data-raw/create_compare_signatures_data.R\` from local
signature RDS files under \`SIGREPO_PATH\` or
\`~/research/projects/sigRepo\`.
