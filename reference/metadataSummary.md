# Summarize collection metadata

Summarize collection metadata

## Usage

``` r
metadataSummary(x, only_shared = TRUE)

# S4 method for class 'OmicSignatureCollection'
metadataSummary(x, only_shared = TRUE)
```

## Arguments

- x:

  An \`OmicSignatureCollection\` object.

- only_shared:

  Use \`TRUE\` to only return shared metadata fields.

## Value

A matrix summarizing metadata fields.
