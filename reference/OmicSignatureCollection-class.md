# OmicSignatureCollection S4 class

Stores metadata and a list of \`OmicSignature\` objects.

## Usage

``` r
# S4 method for class 'OmicSignatureCollection'
show(object)
```

## Arguments

- object:

  An object to display.

## Slots

- `metadata`:

  A list describing the collection.

- `OmicSigList`:

  A list of \`OmicSignature\` objects.
