# Simulated signatures with only uni-directional signatures

A named list of three simulated uni-directional \`OmicSignature\`
objects (no \`group_label\` contrast), designed to demonstrate
\`compare_omic_signatures()\`'s flat, single-comparison output for
\`method = "overlap"\` when none of the compared signatures has
group_label levels to pair on.

## Usage

``` r
data(compare_unidirectional_example)
```

## Format

A named list with three \`OmicSignature\` objects:

- uni_x:

  Simulated uni-directional signature, genes 001-010.

- uni_y:

  Simulated uni-directional signature, genes 006-015.

- uni_z:

  Simulated uni-directional signature, genes 050-059.

## Source

Generated by \`data-raw/create_compare_unidirectional_data.R\`.

## Details

\`uni_x\` and \`uni_y\` share 5 of their 10 features each; \`uni_z\`
shares nothing with either.
